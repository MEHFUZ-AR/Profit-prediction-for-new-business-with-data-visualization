This application can be used to make profit prediction for the new entepreneurs while starting a new business with improved visualization of the results and dataset.
This was developed using python and some of its useful libraries like numpy,pandas,sklearn,matplotlib e.t.c.

* 50_Startups file in this folder is the dummy dataset used for this application.
* To run this application downlaod the StartupProfitPrediction file in this folder with a python installed PC.
